## Description - Descrição

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation - Instalação

```bash
$ npm install
```

## Running the app - Executando o aplicativo

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
